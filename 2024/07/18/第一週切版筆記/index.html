第一週切版筆記

快捷鍵小技巧

  • 快速移動我要移動的 code 到上一層或是下一層可以用 alt+上或是下
    • mac 裡面是 option+上或下
  • 如果要用案件一次選兩行 ⇒ 使用 shift+上下鍵
    • 如果又要移動就是放開後 alt+上或是下鍵
  • 容器通常不寫死高度,而且通城使用容器來定位

container 的用處

  • 是要定位使用的,他是一個容器拿來定位使用

容器(container)與內容(main)分離是蠻重要的！！！！

line-height

字體的高度是來自於行高
可以在全域中作以下設定

  • 內文行高可以統一設定 1.5
  • 標題行高,可以統一設定 1.2
